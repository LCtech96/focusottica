# Focus Ottica - Sito Web

Sito web elegante per Focus Ottica, ottica storica di Castellammare del Golfo (TP) con 25 anni di esperienza.

## Tecnologie

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React (icone)

## Installazione

```bash
npm install
```

## Sviluppo

```bash
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) nel browser.

## Build

```bash
npm run build
npm start
```

## Caratteristiche

- Design elegante in stile iOS
- Richiami al mare siciliano
- Integrazione social media (Facebook, Instagram)
- Pulsanti WhatsApp per contatti diretti
- Mappa Google Maps
- Funnel di conversione ottimizzato
- Design responsive
